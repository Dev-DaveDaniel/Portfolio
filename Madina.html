Madina

This is a hub stytle map where I had to make:

  • Parkour map
  • Parkour character
  • Collectibles

Hub Based Map: this style of map allows the player quick access to each location while enabling designers to have a local gameplay space to have scriptable event take place.

Parkour Character: I wanted to make a parkour character that was able to traverse the map, so I made a couple of parkour functions using the new unreal engines motion warping tool.

  • Ledge Climb
  • Ledge Jump
  • Calculated Jump
  • Slide

Collectibles: I placed collectibles around the map to increase player exploration, this also serves as a reward for the explorer archtype of players and increases engagement in the seeking of these collectibles
